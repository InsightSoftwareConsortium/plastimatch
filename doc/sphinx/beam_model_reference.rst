Beam model reference
--------------------
The beam model is described in a file 
that uses uses the "ini file" format.  

Sections
========
There are five allowed sections.  Each section 

.. list-table::
   :widths: 20 80
   :header-rows: 1

   * - Section
     - Description
   * - [COMMENT]
     - Settings within a COMMENT section are ignored.
   * - [TBD]
     - TBD.
